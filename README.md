# Portfolio
Este repositório reúne um projeto web desenvolvido como parte do meu portfólio, com foco em arquitetura front-end, organização de layout, interações ricas em JavaScript e atenção a detalhes de experiência do usuário.

O projeto explora soluções práticas para interfaces complexas, incluindo grids avançados com CSS Grid, layouts responsivos, componentização visual, manipulação direta do DOM, e interações sem dependência de frameworks. Cada decisão de código prioriza clareza, manutenibilidade e comportamento previsível da interface.

O objetivo deste repositório é demonstrar minha capacidade de planejar interfaces, resolver problemas reais de layout e UX, e escrever código limpo e escalável, servindo como base para experimentação, evolução e adaptação para diferentes contextos de produto.
